# Test for Tutors Roblox

Este repositório contém minha atividade prática desenvolvida no **Roblox Studio** para o processo de seleção de tutores.

## 📌 Objetivo
Demonstrar domínio de construção de mapa, organização de cena e implementação de scripts de jogabilidade.

## ✅ O que foi desenvolvido

- Sistema de **moeda coletável** com contagem em `Coins` (leaderstats).
- **Animação da moeda** com rotação contínua.
- **Plataforma móvel** entre duas ilhas, com ida e volta.
- **Tempo de espera de 8 segundos** para embarque/desembarque em cada ilha.
- Implementação de **dois boosts na loja**:
  - Boost de velocidade
  - Boost de pulo
- Construção e ambientação com:
  - uso da **Caixa de Ferramentas**
  - **arco de lama**
  - **arco com Parts**
  - **saliência em terreno plano**

## 📁 Estrutura do repositório

- `projeto/` → arquivo principal do Roblox Studio (`Test for tutors Roblox.rbxl` / `.rbxlx`)
- `documentos/` → documento com o que foi solicitado na atividade

## ▶️ Como executar

1. Baixe ou clone este repositório.
2. Abra o arquivo do projeto na pasta `projeto/` no Roblox Studio.
3. Execute em modo Play para testar as funcionalidades.
